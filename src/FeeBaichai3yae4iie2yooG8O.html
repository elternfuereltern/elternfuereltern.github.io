Anmeldung im Chat

Bitte beachten Sie die folgenden Hinweise:

  • Im Feld Name können Sie Ihren gewünschen kurzen Benutzername eingeben.
  • Geben Sie im letzten Feld Info unbedingt die Klasse und den Namen der Lehrerin ein, damit wir Sie zuordnen können.
  • Sie bekommen nach der Anmeldung eine Bestätigungsmail; zum Abschließen der Anmeldung müssen Sie auf den darin enthalten Bestätigungslink klicken.

Zur Anmeldung im Chat rufen Sie nun das Registrierungsformular auf.

Wir wünschen Ihnen viel Spaß ... bleiben Sie gesund und freundlich!
